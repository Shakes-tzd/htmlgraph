Description

reindexWorktree calls exec.Command(os.Executable(), 'reindex') which spawns the full Go binary. In tests, os.Executable() returns the test binary, not htmlgraph, causing it to hang waiting on stdin for 10 minutes. Add context.WithTimeout(30s) to fix.
