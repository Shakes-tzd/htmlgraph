Description

plan create and plan generate output the standard node.gohtml template (27-44 lines, plain li steps) instead of the interactive CRISPI template with dagre-d3 dependency graph, approval checkboxes, decision radios, critique panels, finalize preview, and progress bar. The set-section/add-question commands expect PLAN_* placeholders that don't exist in the standard template. The interactive template at cmd/htmlgraph/templates/plan-template.html must be wired into plan create/generate so that plans are human-reviewable. Benchmark: plan-b06fe2db.html (1148 lines, full interactive UI).
