Description

Integrate Claude Code session transcripts into HtmlGraph for rich development history tracking. ## Overview Claude Code stores conversation transcripts as JSONL files in ~/.claude/projects/. These contain: - User messages and assistant responses - Tool calls (Read, Write, Edit, Bash, etc.) - Timestamps and session metadata - Git branch context This feature integrates these transcripts into HtmlGraph to: 1. Link development sessions to tracked features 2. Provide rich context for handoffs between agents 3. Enable analytics on AI coding sessions 4. Create human-readable HTML archives of sessions ## Integration Points - **Import**: Convert JSONL transcripts to HtmlGraph session nodes - **Linking**: Associate sessions with features via git branch/commit - **Export**: Generate HTML transcripts compatible with Simon Willison's claude-code-transcripts format - **Analytics**: Token usage, tool call patterns, session duration ## References - Simon Willison blog post: https://simonwillison.net/2025/Dec/25/claude-code-transcripts/ - claude-code-transcripts tool: https://github.com/simonw/claude-code-transcripts ## Technical Notes - JSONL location: ~/.claude/projects/[encoded-path]/[session-uuid].jsonl - Path encoding: Forward slashes replaced with hyphens - Each line is a JSON object with type, message, uuid, timestamp, sessionId
