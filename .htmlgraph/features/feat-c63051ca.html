Description

The session-start hook should compare the CLI binary's compiled version against the plugin's plugin.json version. If they differ, emit a warning in the hook's AdditionalContext so the user sees it at session start. The CLI binary already has the version embedded via ldflags. The plugin version is in CLAUDE_PLUGIN_ROOT/../../.claude-plugin/plugin.json (or resolve from the marketplace cache). Warning format: 'HtmlGraph version mismatch: CLI v0.48.1 vs plugin v0.47.0. Run htmlgraph build or update the plugin.'
