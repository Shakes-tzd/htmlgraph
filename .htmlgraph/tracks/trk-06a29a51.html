Project Isolation — Prevent Cross-Project Contamination

Todo High Priority

Implementation Steps

  1. ⏳ ARCHITECTURE: Each project with .htmlgraph/ is an independent namespace. Work items, sessions, and events must never leak between projects.
  2. ⏳ ARCHITECTURE: Session-start records the canonical project_dir in both the session DB row and .active-session file. All downstream hooks validate against this.
  3. ⏳ ARCHITECTURE: Claude Code transcripts and git repo identity (remote URL or root path) are the ground truth for project identity — not walk-up heuristics.
  4. ⏳ ARCHITECTURE: When CWD diverges from session project_dir, hooks warn and refuse to write work items to the wrong project.
  5. ⏳ ARCHITECTURE: htmlgraph ingest only imports sessions whose git remote matches the current project.
