Phoenix Dashboard Pythonx Integration

Planned High Priority Created: 2026-03-15

Replace raw SQLite queries in Phoenix LiveView with HtmlGraph Python SDK via Pythonx (embedded CPython in BEAM). Single server, sub-millisecond latency, automatic Elixir-Python data conversion.

Overview

Wire Phoenix LiveView to call HtmlGraph Python SDK directly via Pythonx NIF. Eliminates the need for a separate FastAPI server or stdio bridge. The SDK runs in-process with automatic data type conversion.

Context

The Phoenix dashboard currently duplicates query logic in Elixir (raw SQLite). Pythonx embeds CPython via NIF, giving direct access to the full HtmlGraph SDK (GraphManager, step attribution, edge queries) with ~0.1ms call overhead.
